# Custom Wastebin

A customizable wastebin widget for KDE Plasma 6 that lets you set your own icons for empty and full states.

## Features

- Configurable empty and full trash icons
- Option to use symbolic icons in panels
- Drag-and-drop files to trash
- Right-click menu: Open, Empty, Trash Settings

## Installation

```bash
./install.sh
```

Then restart Plasma or log out/in:
```bash
kquitapp6 plasmashell && kstart plasmashell
```

Add the widget by right-clicking your panel → Add Widgets → search "Custom Wastebin".

## Configuration

Right-click the widget → Configure Custom Wastebin → click the icon buttons to choose icons from your theme.

## Uninstallation

```bash
./uninstall.sh
```
